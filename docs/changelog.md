## [2.1.0] - 2025-08-15

### Додано
- **POS Інтерфейс** (`/pos`)
  - Швидкий пошук товарів по назві, категорії, SKU
  - Корзина замовлення з підрахунком та управлінням кількістю
  - Вибір клієнта для замовлення
  - Оформлення замовлення з автоматичним створенням
  - Повноекранний інтерфейс для зручності роботи

- **Система управління замовленнями** (`/orders`)
  - Перегляд всіх замовлень з детальною інформацією
  - Фільтрація по статусу, клієнту, даті
  - Оновлення статусів замовлень
  - Детальний перегляд замовлення з товарами
  - Експорт замовлень в CSV формат
  - Статистика по замовленнях

- **Управління клієнтами** (`/customers`)
  - CRUD операції з клієнтами
  - Програма лояльності (бонусні бали)
  - Пошук та фільтрація клієнтів
  - Експорт бази клієнтів
  - Статистика по клієнтах

- **Нова схема бази даних**
  - Таблиця `customers` для клієнтів
  - Таблиця `orders` для замовлень
  - Таблиця `order_items` для товарів замовлення
  - RLS політики для безпеки
  - Тригери для автоматичного оновлення сум
  - Функції для генерації номерів замовлень

### Змінено
- Оновлено навігацію в DashboardLayout
  - Додано "POS Система" як основний пункт меню
  - Змінено іконку для "Замовлення" на Receipt
  - Оновлено описи пунктів меню
- Оновлено структуру навігації в документації
- Покращено організацію компонентів

### Технічні деталі
- Створено нові TypeScript типи для POS системи
- Додано компоненти: POSInterface, OrderManagement, CustomerManagement
- Оновлено сторінки: `/pos`, `/orders`, `/customers`
- Інтегровано з існуючою системою аутентифікації та бізнес-логікою
